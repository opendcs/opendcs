# References

_Example images that could assist with design of a modern DCPMon_

## Breakdown

### DCPMon

- App created by CoveSoftware built into LRGS to display static HTML files for status information of DCP

### DCPStatus

- App created by USACE Tulsa District that renders JSON files created by a backend python script reading both OpenDCS XML database and `DcpMessageStatus` (Script within LRGS to read data from the archive given a netlist)
